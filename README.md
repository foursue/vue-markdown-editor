# Vue.js Markdown Editor Sample

Vue.jsの学習用に作った最小のMarkdownEditorです。
ほぼVue.js公式の例のままです。
https://jp.vuejs.org/v2/examples/index.html

# Usage

1. Open index.html
1. input left pane
1. you can find markedown

- [Vue.js](https://jp.vuejs.org/)
- [Lodash](https://lodash.com)
